# Ferret Health Report Generator

## Overview

The Ferret Health Report Generator is a Java application that processes ferret health information and generates organized health reports. The project demonstrates object-oriented programming principles, data processing, and class-based design.

---

## Features

- Stores ferret information
- Calculates health metrics
- Generates organized reports
- Uses multiple Java classes
- Demonstrates object-oriented programming

---

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Classes & Objects
- Methods
- File Processing
